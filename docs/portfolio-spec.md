# Portfolio Website Specification

## 1. Mission

Build a world-class personal portfolio website for Senthil Royan that helps secure Senior Product Designer, Lead Product Designer, Product Design Manager, Principal Product Designer, UX Lead, and AI Product Designer opportunities worldwide.

The site must:
- communicate credibility quickly
- make the product design work feel premium and strategic
- create a strong path to contact, resume download, and professional networking
- position Senthil as a senior design leader, not just a visual designer

---

## 2. Design Philosophy

The visual direction should feel modern, restrained, and premium.

### Design language
- Apple-inspired clarity
- Linear-inspired structure
- Stripe-inspired polish
- Vercel-inspired speed
- Raycast-inspired detail
- Notion-inspired simplicity

### Core qualities
- Modern
- Minimal
- Premium
- Elegant
- Corporate
- Fast
- Confident

### Avoid
- flashy gimmicks
- template-like layouts
- overdesigned agency aesthetics
- heavy motion that hurts usability

---

## 3. Technology Rules

The site should be built with:
- Next.js
- TypeScript
- TailwindCSS

### Constraints
- No GSAP
- No Three.js
- No heavy animation libraries
- Production-ready architecture
- SEO-ready metadata
- Vercel-ready deployment

### Implementation expectations
- component-driven structure
- responsive layouts by default
- accessible interactions
- optimized assets and fonts

---

## 4. Color System

Use a disciplined system of design tokens to ensure consistency across the experience.

### Core tokens
- Primary: `#0F172A` (deep slate)
- Secondary: `#1E293B` (cool slate)
- Accent: `#2563EB` (vibrant blue)
- Accent Soft: `#60A5FA`
- Background: `#F8FAFC`
- Surface: `#FFFFFF`
- Border: `#E2E8F0`
- Text Primary: `#0F172A`
- Text Secondary: `#475569`
- Muted: `#94A3B8`

### Interactive states
- Button Primary: accent blue with strong contrast
- Button Secondary: neutral surface with subtle border
- Hover states: slightly elevated, not flashy

### Dark mode
- Dark background: `#020617`
- Elevated surface: `#0F172A`
- Text: `#F8FAFC`
- Accent remains vivid but slightly softened

---

## 5. Typography

Typography should feel editorial, modern, and highly readable.

### Type scale
- H1: 56-72px desktop, 40-48px mobile
- H2: 36-44px desktop, 28-32px mobile
- H3: 24-30px desktop, 20-24px mobile
- H4: 18-22px desktop, 16-20px mobile
- Body: 16-18px
- Small: 14px

### Typography rules
- Heading weight: 600-700
- Body weight: 400-500
- Letter spacing: tight for headings, relaxed for body
- Line height: 1.1 for headlines, 1.6 for body copy
- Responsive typography: scale down smoothly on smaller screens

---

## 6. Layout System

Use a clear and modular grid system.

### Layout rules
- Max content width: 1200px
- Section padding: 96px desktop, 64px tablet, 48px mobile
- Container padding: 24px mobile, 32px tablet, 40px desktop
- Grid: 12-column on desktop, 8-column on tablet, 4-column on mobile

### Breakpoints
- Mobile: 0-767px
- Tablet: 768-1023px
- Desktop: 1024px+

### Spacing system
- 8px base unit
- Use consistent spacing rhythm across all sections

---

## 7. Hero Section

The hero must immediately establish seniority, clarity, and momentum.

### Required elements
- Badge: “Senior Product Designer” or similar
- Headline: concise, outcome-led, and premium
- Supporting description: explain design leadership and product impact
